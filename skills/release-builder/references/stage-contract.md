# 发布技能阶段契约

## 输入

- 已确认的 `Product-Spec.md`
- `DEV-PLAN.md`
- 实施证据和审查结果
- 仓库构建、打包和部署配置
- 目标发布环境说明

## 输出

- 发布准备报告
- 隐私审计结果
- 构建、打包或部署证据
- 阻塞项和需要用户处理的动作

## 合格标准

发布准备必须验证：

- 没有明显敏感信息暴露
- 配置和秘密来源被安全说明
- 构建或打包路径可运行，或阻塞明确
- 部署步骤可复现
- 必要时有回滚或恢复路径

## 禁止发布的情况

- 真实秘密被提交或打印
- 用户数据处理不清楚
- 构建或打包失败
- 部署依赖未记录的手工状态
- 计划要求的测试或审查门禁缺失
